# 输入框快捷键优化

**完成日期**：2026-03-08

## 功能描述

优化了聊天输入框的快捷键行为，使其更符合现代聊天应用的使用习惯：
- 直接按 Enter 键发送消息
- 按 Shift + Enter 键实现换行

## 实现细节

### 修改的文件

1. `src/App.vue`
   - 将 `@keyup.enter.ctrl` 改为 `@keydown.enter`
   - 添加 `handleEnterKey` 函数处理键盘事件

2. `src/i18n/en.ts`
   - 更新英文提示文本：`Type a message, press Enter to send`

3. `src/i18n/zh.ts`
   - 更新中文提示文本：`输入消息，按Enter发送`

### 关键代码

```typescript
// 处理回车键
const handleEnterKey = (event: KeyboardEvent) => {
  // Shift + Enter: 换行（默认行为，不阻止）
  if (event.shiftKey) {
    return
  }
  
  // 单独 Enter: 发送消息
  event.preventDefault()
  sendMessage()
}
```

### 技术要点

- 使用 `@keydown.enter` 而不是 `@keyup.enter.ctrl`，确保在按键按下时就能捕获事件
- 通过 `event.shiftKey` 判断是否同时按下 Shift 键
- 使用 `event.preventDefault()` 阻止默认的换行行为

## 使用方法

在聊天输入框中：
1. 输入消息后，直接按 **Enter** 键即可发送
2. 如需换行，按 **Shift + Enter** 组合键

## 注意事项

- 该功能需要刷新页面或重启应用后生效
- 适用于所有支持的语言界面
